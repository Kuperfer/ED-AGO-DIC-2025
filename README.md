# ED-AGO-DIC-2025
Descripción general:
Este repositorio reúne las prácticas y proyectos desarrollados a lo largo del curso "Estructura de Datos" por Fernando Alonso Ku Pérez. Contiene implementaciones en C de estructuras de datos, algoritmos y simulaciones (pilas, listas enlazadas, árboles, conversión y evaluación de expresiones, simulación de colas, algoritmos de ordenamiento, etc.). Está pensado como un portafolio técnico y como base para compilar, ejecutar y estudiar cada práctica de forma independiente.

Proyectos incluidos:
- Conversor Infijo a Postfijo y Evaluador Convierte expresiones aritméticas infijas a notación postfija y evalúa el resultado; entrada por archivo y salida en exp_postfijas.txt.

- Simulación de Atención en Bancos Simula tres colas atendidas por hilos; genera tiempos de llegada y duración de transacciones aleatoriamente.

- Lista de Movies con Lista Doblemente Enlazada CRUD sobre un CSV de películas; incluye manejo de lista doblemente enlazada y utilidades para Movie.

- Sistema de Gestión de Alumnos con ABB Árbol binario de búsqueda para almacenar alumnos, calificaciones dinámicas, reportes y visualización del árbol.

- Una breve presentación hacerca del método de ordenamiento "Shell", junto a su respectiva investigación.

- Colección de Algoritmos de Ordenamiento Carpetas por equipo con implementaciones en C de Bubble, Quick, Heap, Insertion, Shell, Tournament, etc., y guía de trabajo en equipo.

- Makefile y Automatización Makefile para compilar y ejecutar prácticas seleccionadas en entornos MSYS2 MINGW64 (Windows) y Unix-like cuando aplique.


Requisitos y compilación:
- Lenguaje: C (estándar compatible con gcc/clang)

- Dependencias: biblioteca matemática -lm cuando sea necesario; make para proyectos con Makefile; MSYS2 MINGW64 recomendado en Windows.

- Notas: Asegúrate de colocar los archivos de entrada (por ejemplo exp_infijas.txt, Movies.csv) en el mismo directorio del ejecutable según lo indique cada práctica.
